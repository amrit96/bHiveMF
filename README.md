# bHiveMF

For Local run:
1. Set up local sql db
2. set up environment and .env file.
3. .env file to have the following keys
    DB_HOST=127.0.0.1 
    DB_USER=root
    DB_PASSWORD= <your db password>
    DB_NAME=mutual_fund
    DB_DIALECT=mysql
    JWT_SECRET= <your jwt secret>
    RAPIDAPI_KEY= <your rapidapi key>
4. Run server : npm run dev
